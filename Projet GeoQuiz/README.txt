### Installation de la bibliothèque Basemap :

- Aller dans le terminal de votre IDE 
- Tapez : pip install Basemap  (si vous avez plusieurs versions de python, tapez : pip3 install Basemap)
- Appuyez sur Entrée  

# Voici une liste de commande pouvant aider si vous rencontrez des problèmes :

- conda install -c conda-forge basemap
- conda install -c conda-forge/label/gcc7 basemap

### Pour lancer le programme :

- Ouvrez le dossier contenants les fichiers du projet dans votre IDE
- Lancez le fichier main.py
